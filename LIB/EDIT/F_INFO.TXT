# File: f_info.txt


# This file is used to initialize the "lib/raw/f_info.raw" file, which is
# used to initialize the "terrain feature" information for the Angband game.

# Do not modify this file unless you know exactly what you are doing,
# unless you wish to risk possible system crashes and broken savefiles.

# Note that the terrain feature are grouped into very regular groups,
# such that each of the bits in the feature type conveys information.

# Note that terrain feature zero contains the "darkness" picture.

# GSNband notes:

# I added Kamband-ish chaos fog and the six altars (there were as many as
# 26 in versions before 1.0). Note that there are three altar colours
# (two gods per each colour). You are not supposed to know beforehand
# what the god of the altar is.

# I also added trap names to their outward descriptions. Ideally, you
# should be able to distinguish a particular trap only after you've 
# experienced the effects.

# This is still unstable ground and very likely to change in future.

# Version stamp (required)

V:2.8.1
