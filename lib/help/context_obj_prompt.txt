<style:title>Selecting Objects</style>

Objects are a big part of playing <color:keyword>FrogComposband</color>. Many commands (such as quaffing potions or reading scrolls) require you to select an object to use. Other commands allow you to view and inspect the objects you are carrying or using. This helpfile explains the user interface for viewing and selecting objects.

<topic:Sample><style:heading>An Example</style>

The top line shows the available tabs (Inventory, Equipment and Quiver in this case) and the active tab is displayed in <color:G>color</color> while the inactive tabs are <color:D>grayed out</color>. You can switch tabs by pressing <color:keypress>/</color>.

To choose an object, simply press the indicated letter. What happens next depends on the context of the object prompt and is indicated by the bottom line on the dialog. In this case, you are simply browsing your gear, so what could be more useful than letting you inspect individual objects?

You'll notice that selecting an object in this case did not dismiss the dialog. This is both unusual and useful, since you might want to inspect multiple objects before you are done. A few other game commands behave this way, such as Inscribing (<color:keypress>{</color>) objects, but most dialogs are dismissed after a selection is made.

<topic:QuickRef><style:heading>Quick Reference</style>
  <indent>
<color:G>Selection</color>
<color:keypress>a-z</color>       Select Indicated Object
<color:keypress>A-Z</color>       Inspect Indicated Object
<color:keypress>Page Up</color>   Scroll Through Long Object Lists
<color:keypress>Page Down</color>
<color:keypress>Space</color>
<color:keypress>Esc</color>       Cancel the Prompt

<color:G>Tab Navigation</color>
<color:keypress>/</color>         Next Tab
<color:keypress>\</color>         Previous Tab
<color:keypress>^P</color>        Switch to Pack (Inventory) Tab
<color:keypress>^E</color>        Switch to Equipment Tab
<color:keypress>^Q</color>        Switch to Quiver Tab
<color:keypress>^F</color>        Switch to Floor Tab
<color:keypress>- </color>        <indent>Switch to Floor Tab. Select Floor Object if there is only one.</indent>

<color:G>Display Options</color>
<color:keypress>^W</color>        Toggle Weights (see <link:option.txt#show_weights>)
<color:keypress>^G</color>        Toggle Graphics (see <link:option.txt#show_item_graph>)
<color:keypress>^D</color>        Toggle Slot Descriptions (see <link:option.txt#describe_slots>)
<color:keypress>@</color>         Toggle Inscription Label Overrides (Default On)

</indent>
