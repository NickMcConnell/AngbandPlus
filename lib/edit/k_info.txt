# is the event number that is going to be called by the lua script. The "misc" attribute

# 1: Intelligence

# 4: Constitution

# extra1 = type of damages. 0 defaults to Physical.

##### Chests #####

# The chests have been redesigned as of Portralis 0.4.
# Though they have the SPECIAL_GENE flag, they now have their own code
# to be randomly placed on any dungeon levels.

# Wooden chests can be opened by anyone, but contains the lowest quality items.
# Iron chests are locked, and lockpicking must be attempted. Any characters can attempt it, and they contain better items than wooden chests.
# Steel chests are much more difficult to open than Iron Chests, but contain high quality goods.

# From this point, all other chests cannot be opened unless the player has the 'Lockpicking' ability of the Rogue class.

# Magical chests contain magic items, usually of higher quality than the current dungeon level.
# Cursed chests contain cursed items.
# Ancient chests contain an artifact.
# Strange chests contain special items, are VERY difficult to open, and also very rare. Only masters of Lockpicking will be able to open them!

# The sval determine what kind of items will drop, and is currently hard-coded.
# The pval is 0 by default, but it changes. It represents the level of the items in the chest. This affect content and it's enchantments.
# The 'extra' represents the difficulty of opening the chest. The basic is a test between the player's dex and int vs (((Chest's pval) * (dun level)) * extra)

# Do not change the index number of the chests. The starting index, in this case 38, is defined in the code and moving the chests definitions
# will break the chests.
