# File: r_info.txt

# This file is used to initialize the "lib/data/r_info.raw" file, which is
# used to initialize the "monster race" information for the Angband game.

# Do not modify this file unless you know exactly what you are doing,
# unless you wish to risk possible system crashes and broken savefiles.
# After modifying this file, delete the "lib/data/r_info.raw" file.

# Currently, "unique" monsters are just "special" monster races, with
# the requirement that only one monster of that race can exist at a time,
# and when it is killed, it can never again be generated.

# The "0" and "9" symbols are reserved for internal debugging use.

# The "&" symbol is reserved for future use as a special "terrain feature".

# The "`" symbol is reserved as an "alternate" open door picture, since
# the "'" symbol looks like the "," symbol in some fonts (esp. IBM).

# As always, you can enforce any "visual picture" you want with a "pref file".

# Note that monster zero contains the "player" picture.


# === Understanding r_info.txt === 

# 2 N:   serial number : monster name
# 2 G:   symbol : color
# 4 I:   speed (code uses this value *5 and +85) : min and max group
#        size : vision (currently all are at 50) : alertness (currently all are
#        at 5)
# 3 W:   act (0-5, 0 = town/cow-level) : rarity (most are at 1,
#        escorts/minions/etc are at 100, also used 2 and 5) : r_idx of escort
#        (0 for none)
# 4 JKL: depth : min and max hitpoints : armor class (usual max seems
#        to be 528) : xp (code uses /10 of this value, players need *2 XP, so
#        monsters are worth /20 of what is listed here)
# 3 ABC: attack method : attack effect : min and max damage
# 6 XYZ: dmg, magic, fire, elec, cold, pois (dmg and magic currently
#        have no effect) these can also be negative for a 'susceptibility'
# 3 H:   (unused) element : chance to add to normal attack : min and max damage
#   F:   flag | flag | etc
#   S:   spell frequency |
#   S:   spell type | spell type | etc
#   D:   Description

# Uniques get DROP_3D2, DROP_GOOD and ONLY_ITEM, group leaders get
# DROP_1D2 and ONLY_ITEM, most other monsters get DROP_60
