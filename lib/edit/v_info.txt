# File: v_info.txt


# This file is used to initialize the "lib/raw/v_info.raw" file, which is
# used to initialize the "vault template" information for the Angband game.

# Do not modify this file unless you know exactly what you are doing,
# unless you wish to risk possible system crashes and broken savefiles.

# After modifying this file, delete the "lib/raw/v_info.raw" file.

# Note that the "spacing" in the "description" lines is very important!
# Also note that a dash is equivalent to a space; this is useful
# if you want to use e.g. Emacs picture mode.

# Version stamp (required)


# Kamband 1.2 greatly improved vault generation -- Now shops,
# buildings, town/arena levels, and even quest information
# is stored in this file.
#
# Kamband 1.8 also improved vault generation -- Now terrain features and
# monsters/objects have separate maps.
#
#
# Vault type 10 means a town level, and type 11 means arena level.
# Type 12 is a store layout.
# Type 13 is a wilderness vault.
# One of the arena/town levels will be chosen at birth.
#
# Format:
#
# N:number:name
# X:type:rating:height:width
# Y:monster index:... (up to ten)
# G:generation info
# Q:quest index:quest type:quest kind
# U:quest description...
# D:vault description...
# M:vault monster/object descriptions.
#
#
# Quest type is 1 for ``eliminate'', 2 for ``scout''.
#
# Generation info is 0 for normal, 1 for terrain level, 2 for foggy level.
#
#
# Terrain symbols:
#
# a-z => building
# 0-9 => shop
#
# # => Wall, inner
# % => Wall, outer
# * => Treasure/Trap
# : => Rubble
# & => Magma
# $ => Quartz
# + => Secret door
# ; => Glyph of warding
# < => Stairs up
# > => Stairs down
# ^ => Trap
#
# A => Grass
# B => Swamp
# C => Mud
# D => Door
# E => Quest exit
# F => Chaos fog
# G => Generator of monster 1.
# H => Shrub
# I => Rocky hill
# J => Fog
# K => Shallow lava
# L => Deep lava
# M => Mountain
# (N)
# O => Random altar
# (P)
# Q => Quest entrance
# (R)
# S => Store exit
# (T)
# U => Shaft
# V => Shallow water
# W => Deep water
# X => Wall, permanent
# Y => Trees
# (Z)
#
# Monster/object symbols:
#
# 0-9   => Predefined monster.
#
# a-z, A-Z           => Monster of that letter.
# !"$(),-/=?[\]_{|}  => Object of that letter.
#
#
# Old | New
#
#       * => Treasure/trap
# (T)   . => Treasure
#       & => Monster, possibly the arena monster.
# (,)   : => Monster and/or object
# (8)   ^ => Nasty monster
# (9)   # => Meaner monster + treasure
# (@)   ; => Meaner Monster
# (P)   @ => Player
#
